\doxysubsection{Detailed Description}
Abstract base class which entities are based off of.

Entities are generally anything that can have a sprite and/or a position.

To make your subclass of \mbox{\hyperlink{class_entity}{Entity}} visible on the screen, you need to do the following\+:


\begin{DoxyEnumerate}
\item set \+\_\+texture
\item set \+\_\+sprite
\item Somewhere in the main game loop where drawing happens, call your subclass\textquotesingle{}s \mbox{\hyperlink{class_entity_a820404587abe78b6383d829348efbe2b}{on\+Draw()}} method and draw() the entity\textquotesingle{}s sprite to the screen.
\end{DoxyEnumerate}

Updating position is handled by \mbox{\hyperlink{class_entity_a205443d87b309a074a6088b6e148f2a2}{update()}}. Position is set based on the value of \+\_\+velocity. You can set \+\_\+position or \+\_\+velocity in \mbox{\hyperlink{class_entity_aeb0bd1940e7932d3e244984259b5cc0e}{on\+Update()}} to define the movement behavior of your \mbox{\hyperlink{class_entity}{Entity}}.

\doxysubsubsection{\texorpdfstring{\_sprite}{\_sprite}}
{\footnotesize\ttfamily sf\+::\+Sprite Entity\+::\+\_\+sprite\hspace{0.3cm}{\ttfamily [protected]}}

The sprite that this entity uses.

The active sprite adds a rectangle around a texture, defining the region of the texture that we actually want to display.

To give your subclass a sprite, do something like the following in the constructor after giving your subclass a texture\+: \begin{DoxyVerb}// You've already set _texture, now set _sprite
_sprite.setTexture(_texture);
// set your rectangle, origin, etc. See docs on sf::Sprite.
// Here's an example:
_sprite.setOrigin((int)(_texture.getSize().x / 2), (int)(_texture.getSize().y / 2));
\end{DoxyVerb}
 One use case of this is animation; you could use one texture to store all the frames of an animation and update the sprite to change which part of the texture it renders when you want to show a different animation frame. \mbox{\Hypertarget{class_entity_a8e409f1b737a84cd895e42e4ec8c28bc}\label{class_entity_a8e409f1b737a84cd895e42e4ec8c28bc}}
